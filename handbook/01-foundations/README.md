## ✅ Self-Assessment

Test your understanding:

1. **What is the difference between AI and Machine Learning?**
   <details>
   <summary>Answer</summary>
   AI is the broader concept of machines being able to carry out tasks in a way we would consider "smart." ML is a subset of AI that enables machines to learn from data without being explicitly programmed.
   </details>

2. **Name the three main types of Machine Learning.**
   <details>
   <summary>Answer</summary>
   Supervised Learning, Unsupervised Learning, and Reinforcement Learning.
   </details>

3. **What makes Deep Learning "deep"?**
   <details>
   <summary>Answer</summary>
   The "deep" refers to the multiple layers (depth) of neural networks used to learn increasingly complex patterns.
   </details>

4. **What is the primary focus of an AI Engineer?**
   <details>
   <summary>Answer</summary>
   Building applications using existing AI models, focusing on integration, deployment, user experience, and production systems rather than training models from scratch.
   </details>

## 💡 Key Concepts to Remember

| Concept | Definition | Example |
|---------|------------|---------|
| **AI** | Machines mimicking human intelligence | Chess-playing computer |
| **ML** | Learning from data without explicit programming | Spam filter |
| **Deep Learning** | ML using multi-layer neural networks | Image recognition |
| **Supervised Learning** | Learning from labeled data | Email classification |
| **Unsupervised Learning** | Finding patterns in unlabeled data | Customer segmentation |
| **Reinforcement Learning** | Learning through trial and error | Game-playing AI |
