# GCP Services Handbook

This directory contains comprehensive documentation for all major Google Cloud Platform (GCP) services, including their use cases, resource links, and reference documentation.

## 📁 Directory Structure

### 🖥️ [Compute Services](./compute/)
- **Compute Engine** - Virtual machines and computing infrastructure
- **Google Kubernetes Engine (GKE)** - Managed Kubernetes service
- **Cloud Run** - Serverless container platform
- **App Engine** - Platform as a Service (PaaS)
- **Cloud Functions** - Serverless functions

### 🗄️ [Storage Services](./storage/)
- **Cloud Storage** - Object storage for unstructured data
- **Cloud SQL** - Managed relational databases
- **Firestore** - NoSQL document database
- **Cloud Bigtable** - Wide-column NoSQL database
- **Cloud Spanner** - Globally distributed relational database

### 🌐 [Networking Services](./networking/)
- **VPC Network** - Virtual Private Cloud networking
- **Cloud Load Balancing** - Traffic distribution
- **Cloud CDN** - Content Delivery Network
- **Cloud DNS** - Domain name system
- **Cloud Interconnect** - Private connectivity

### 🤖 [AI & ML Services](./ai-ml/)
- **Vertex AI** - Unified ML platform
- **AutoML** - Automated machine learning
- **AI Platform** - ML training and prediction
- **Speech-to-Text** - Speech recognition
- **Natural Language API** - Text analysis
- **Vision API** - Image analysis

### 📊 [Data & Analytics](./data-analytics/)
- **BigQuery** - Data warehouse and analytics
- **Dataflow** - Stream and batch data processing
- **Dataproc** - Managed Apache Spark and Hadoop
- **Pub/Sub** - Messaging service
- **Data Studio** - Business intelligence
- **Looker** - Data platform

### ⚙️ [Management Tools](./management/)
- **Cloud Console** - Web-based management interface
- **Cloud SDK (gcloud)** - Command-line tools
- **Cloud Monitoring** - Performance monitoring
- **Cloud Logging** - Log management
- **Cloud Debugger** - Application debugging
- **Deployment Manager** - Infrastructure as code

### 🔒 [Security Services](./security/)
- **Identity and Access Management (IAM)** - Access control
- **Security Command Center** - Security monitoring
- **Cloud KMS** - Key management service
- **Secret Manager** - Secrets management
- **Cloud Armor** - DDoS protection
- **Resource Manager** - Resource organization

## 🚀 Getting Started

Each service directory contains:
- **README.md** - Overview and getting started guide
- **QUICK_REFERENCE.md** - Quick commands and common operations
- **SETUP.md** - Detailed setup and configuration guide
- **USE_CASES.md** - Real-world use cases and examples
- **requirements.txt** - Required Python packages (if applicable)
